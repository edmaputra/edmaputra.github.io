Oracle Certified Associate

24-02-2019

Java Oracle OCA Certification 1Z0-808

Setiap Software Engineer pasti ingin mempunyai sertifikat khususnya sertifikat internasional sesuai dengan keahlian yang dimilikinya. Tujuannya tidak lain agar punya nilai tambah sehingga dapat lebih mudah dalam mencari pekerjaan. Kali ini saya ingin berbagi pengalaman mengenai bagaimana proses yang saya tempuh hingga akhirnya saya diakui sebagai Oracle Certified Associate Java SE 8 Programmer.

Pada saat itu tepatnya bulan Juli 2018 saya ingin ikut sertifikasi Java 7 SE Programmer dengan kode ujian Oracle 1Z0-803, tapi setelah mencari informasi, ternyata sertifikat ini akan retired pada akhir tahun 2018. Jadi saya akhirnya mengambil ujian Java SE 8 Programmer (1Z0-808). Di situs Education Oracle ini juga sudah dijelaskan bahwa jika ingin mendaftar ujiannya bisa langsung ke PearsonVue. Di Indonesia sendiri sudah banyak tempat atau lembaga yang terdaftar sebagai tempat ujian Pearson Vue. Perlu diketahui juga bahwa ujian ini sifatnya online, jadi soal yang akan kita kerjakan nanti adalah soal langsung dari Oracle sendiri.

Ada beberapa cara yang bisa dilakukan agar bisa mendaftar ujian ini :

  • Mengikuti program persiapan ujian, setelah itu langsung mengikuti ujiannya. Biasanya program ini diselenggarakan oleh lembaga-lembaga yang memang khusus dalam sertifikasi IT seperti Inixindo, Brainmatics, dan sebagainya. Program ini tidak reguler, biasanya lembaga tersebut akan menentukan kapan mereka menyelenggarakannya sambil mengisi kuota peserta yang akan mengikuti program tersebut. Cara ini lebih mahal jika dibandingkan dengan langsung mengikuti ujiannya karena ada biaya untuk persiapan terlebih dahulu.
  • Langsung mengikuti ujian di lembaga sertifikasi IT yang telah disebutkan sebelumnya. Dengan cara ini lembaga akan mengurus segala administrasi untuk mengikuti ujian ini dan kita sebagai peserta tinggal membayar biaya ujiannya, serta kita dapat menentukan jadwal kapan kita akan mengikuti ujian tersebut di lokasi dimana lembaga tersebut berada.
  • Langsung urus sendiri. Mulai dari pembayaran, lokasi ujian, jadwal mengikuti ujian, bisa kita lakukan sendiri. Dengan ini kita dapat lebih bebas untuk memilih tempat dan waktu kapan kita ujian, hanya saja perlu Kartu Kredit untuk membayar ujian tersebut di Pearson Vue

Saya memilih cara yang terakhir saya sebutkan. Saya bayar sendiri, kemudian saya bisa menentukan tanggal dan jam kapan saya akan mengikuti ujian. Tempat ujian adalah hal yang paling sulit saya tentukan, karena saat itu saya berdomisili di Kabupaten Bulungan, Kalimantan Utara. Saya juga berasumsi bahwa hanya di Pulau Jawa sajalah tempat ujian yang tersedia. Namun setelah saya mencari informasi, ternyata ada satu tempat ujian yang terdaftar di Balikpapan. Setidaknya saya bisa mengurangi ongkos transport hehe. Untuk ujian ini, biaya untuk mengikutinya adalah USD 150 (September 2018).

Untuk daftarnya di PearsonVue, buat akun dulu, kemudian tentukan ujian yang ingin diikuti, tentukan tanggal dan tempat kita akan melaksanakan ujian, setelah itu langsung bayar.

Hal yang dilakukan selanjutnya adalah belajar. Orang-orang yang telah lulus ujian ini dari berbagai negara berbagi pengalaman, tip, trick di Forum Java coderanch.com dan sebagian besar membaca buku OCA Study Guide karya Jeanne Boyarsky dan Scott Selikoff, serta Mock Test 1Z0-808 dari Enthuware. Mock test yang dijual oleh Enthuware ini sangat murah jika dibandingkan dengan mock test lainnya, hanya sekitar USD 10-15 saja.

Saya mengikuti ujian ini pada bulan September 2018. Ujian ini jumlah soalnya 70 multiple choice dengan durasi 150 menit, target nilai lulus ada 65%. Jujur saat itu kondisi saya tidak 100%, mata saya kering, serta saya harus berangkat subuh dari Tarakan ke Balikpapan. Pada saat ujian dimulai, warna backgroundnya putih sehingga sedikit menyilaukan mata saya. Sekitar 15-20 menit baru saya bisa benar-benar berkonsentrasi dalam mengerjakan soal tersebut. Setelah selesai, saya menunggu beberapa jam sampai hasil dari ujian tersebut keluar sambil berdoa. Alhamdulillah saya bisa lulus dengan nilai 85%.

Ada beberapa hal yang harus diperhatikan sebelum dan saat mengikuti ujian ini.

  • Berdoa, tentu saja
  • Sering-sering belajar dan latihan soal. Terbukti memang latihan soal ini sangat membantu apalagi jika ini adalah pengalaman pertama teman-teman dalam mengikuti ujian sertifikasi
  • Datang lebih awal. Ujian ini sebenarnya saya jadwalkan jam 13.00, tapi bisa dimajukan (saya majukan menjadi jam 11.00) jika kita sudah benar-benar siap.
  • Tidur yang cukup serta jaga kesehatan sebelum mengikuti test, agar bisa lebih berkonsentrasi.

Demikian pengalaman saya ini. Semoga teman-teman juga bisa ikut ujian ini maupun ujian sertifikasi lainnya sesuai dengan keahlian masing-masing agar bisa meningkatkan kompetensi serta menjadikan Indonesia sebagai salah satu ladang Software Engineer handal di dunia.

Salam.
